HT For Web架构


#### 安装教程

1. 方便启动，执行命令：npm install -g live-server。
2. 运行run.bat文件
    或：先执行 npm install express 安装依赖；再执行 node server.js 命令即可启动服务。默认端口8099
3. 访问地址：http://localhost:8090/index.html
